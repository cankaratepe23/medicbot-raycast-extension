# MedicBot

Interacts with the Web API of MedicBot to search for audio clips and share them easily.

## Setup
- Create a Discord OAuth app that supports PKCE.
- Set the redirect URI to `https://raycast.com/redirect?packageName=Extension`.
- In Raycast preferences, set the **API Base URL** (defaults to `https://api.medicbot.comaristan.com`).
- In Raycast preferences, set the **Shareable Link Base URL** (defaults to `https://medicbot.comaristan.com`).
- In Raycast preferences, set the **Discord OAuth Client ID**
